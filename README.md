# WriteFlow X

Modern writing and content analysis platform with real-time editor, AI content generation, performance tracking, and interactive dashboards.

## Features

- **Real-time Editor** – Split-screen writing with analysis  
- **AI Content Generation** – Multiple content variations instantly  
- **Analytics Dashboard** – Interactive charts and performance metrics  
- **Voice Consistency Checker** – Maintain brand voice  
- **Pricing & Feature Comparison** – Interactive signup simulation  
- **Responsive & Accessible** – Mobile-first, ARIA support, keyboard navigation  

## Quick Start

1. Clone the repo:  
```bash
git clone <repo-url>
cd writeflow-x
````

2. Open `index.html` for landing page or `app.html` for the editor interface.
